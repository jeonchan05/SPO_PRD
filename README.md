# SPO

SPO(Study Point Online)는 학원과 학생이 함께 사용하는 학습 운영 플랫폼입니다.
학습 계획부터 스터디 실행, 출석/기록, 공지, 피드백까지 한 흐름으로 관리할 수 있도록 설계했습니다.

## 서비스 소개

학원은 학생 운영과 스터디 관리를 체계적으로 진행하고,  
학생은 개인 학습 루틴과 스터디 활동을 한 곳에서 이어서 관리할 수 있습니다.

## 주요 기능

1. 학원 운영 관리: 학생 관리, 공지 등록, 스터디 모집/매칭
2. 스터디룸: 학습 세션 작성, 제출, 주제/기록 관리
3. 출석 및 학습시간: 출석 상태와 학습시간 집계
4. 일정 관리: 캘린더 기반 개인 일정 작성/확인
5. 알림 및 친구 기능: 주요 이벤트 알림과 학습 네트워크 지원
6. 리워드 시스템: 참여도 기반 보상 운영

## 우리가 제공하는 가치

1. 학원 관점: 운영 효율화와 학습 관리 표준화
2. 학생 관점: 자기주도 학습 루틴 형성 및 동기 강화
3. 공통 관점: 학습 데이터를 기반으로 한 투명한 커뮤니케이션
